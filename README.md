# 📊 Competitor Pricing Analysis Dashboard

This project scrapes pricing data from major cloud service providers and performs a comparative analysis using Python. It visualizes pricing trends and generates insights about market positioning.

---

## 🚀 Features

- 🌐 Web scraping of competitor pricing pages
- 🧹 Data cleaning and preprocessing
- 📊 Statistical analysis (mean, min, max, distribution)
- 📈 Visualizations using Matplotlib & Plotly
- 🧠 Basic business insights generation
- 💾 Export results to CSV

---

## 🏢 Competitors Analyzed

- DigitalOcean
- AWS (Amazon Web Services)
- Heroku

---

## 🛠️ Tech Stack

- Python 3.x
- requests
- BeautifulSoup (bs4)
- pandas
- matplotlib
- plotly
